# AirPump5V - Inflate Only

DESCRIPTION

**In this tutorial we will cover how to use a 5V Air Pump to inflate small-scale projects.**

----

WARNING!

- These pumps ***CANNOT be used to pump water!!***

- *For inflate/deflate control, please see the tutorial: AirPump_5V_InflateDeflate(upcoming)*

- *For larger inflating projets, you will need a more powerful Air Pump.*

----

HARDWARE
----

- AIR PUMP MOTOR [(see reference here)](https://thepihut.com/products/air-pump-and-vacuum-dc-motor-4-5v-and-1-8-lpm)
- AIR PUMP BULB
- SILICONE TUBING
- IRF520 MOSFET Driver Module

- Arduino UNO
- LED
- 5V Power Supply
